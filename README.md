# EvolutionaryAlgo-StatArb
Evolutionary Algorithms for Statistical Arbitarege

This is a Machine Learning Project for optimizing technical indicators rules 
for Statistical Arbitrage

The Data Used for this Application are on ETFs 

The parameters that the Algo is optimizing  are a) SMA and b) Bollinger Bands
